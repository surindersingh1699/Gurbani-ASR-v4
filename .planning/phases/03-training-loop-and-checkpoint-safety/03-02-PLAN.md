Add the `HubPushCallback(TrainerCallback)` class to surt/train.py. This callback implements the Hub push strategy from CONTEXT.md decisions.

**Imports to add** (if not already present):
```python
import json

```

**HubPushCallback class:**

Constructor parameters:
- `hub_repo: str` -- Hub repo ID (TRAINING_HUB_REPO from config)
- `processor` -- WhisperProcessor for saving alongside model
- `output_dir: str` -- OUTPUT_DIR, where best_wer.json lives
- `push_every_n_evals: int = 3` -- periodic safety push frequency

Instance variables:
- `self.hub_repo = hub_repo`
- `self.processor = processor`
- `self.output_dir = Path(output_dir)`
- `self.push_every_n_evals = push_every_n_evals`
- `self.eval_count = 0`
- `self.api = HfApi()`
- `self.best_wer_path = self.output_dir / "best_wer.json"`
- `self.best_wer = self._load_best_wer()` -- loads from file if exists (survives resume)

**Methods:**

`_load_best_wer() -> float`:
- If `self.best_wer_path.exists()`, read JSON and return `data.get("best_wer", float("inf"))`
- Otherwise return `float("inf")`
- Print: `[train] Loaded best WER: {wer} from {path}` or `[train] No previous best WER found, starting fresh`

`_save_best_wer(wer: float, step: int)`:
- Write JSON `{"best_wer": wer, "step": step}` to `self.best_wer_path`

`_push_to_hub(model, step: int, wer: float, reason: str)`:
- Create staging directory: `staging_dir = self.output_dir / "hub_staging"`
- `model.save_pretrained(staging_dir)`
- `self.processor.save_pretrained(staging_dir)`
- Build commit message: `f"step {step} | WER {wer:.2f} | {reason}"`
- Call `self.api.upload_folder(repo_id=self.hub_repo, folder_path=str(staging_dir), commit_message=commit_msg)`
- Print: `[train] Hub push ({reason}): {commit_msg}`
- Do NOT use `run_as_future=True` for v1 -- synchronous push is simpler and eval only happens every 300 steps. Accept the blocking time.

`on_evaluate(self, args, state, control, metrics=None, **kwargs)`:
- Increment `self.eval_count`
- Get `current_wer = metrics.get("eval_wer", float("inf"))`
- Get `step = state.global_step`
- Determine `is_best = current_wer < self.best_wer`
- Determine `is_periodic = (self.eval_count % self.push_every_n_evals) == 0`
- If `is_best`: update `self.best_wer`, call `self._save_best_wer(current_wer, step)`
- If `is_best or is_periodic`: call `self._push_to_hub(kwargs.get("model", state.best_model_checkpoint), step, current_wer, "best" if is_best else "periodic")`
  - For the model argument: the `model` is available as `kwargs["model"]` in the `on_evaluate` callback in Seq2SeqTrainer. If not available, fall back to loading from the trainer instance.
- Log: `[train] Eval step {step}: WER={current_wer:.2f} (best={self.best_wer:.2f}, evals={self.eval_count})`

**CRITICAL per CONTEXT.md decisions:**
- Push on WER improvement ("best") PLUS periodic safety push (every 3rd eval regardless of WER)
- Push full model folder (model weights + processor + tokenizer + generation_config) -- Hub repo is instantly loadable with from_pretrained()
- Push to TRAINING_HUB_REPO, NOT HF_MODEL_REPO
- best_wer.json in OUTPUT_DIR survives resume cycles

Step 9 -- Create trainer:

    train_dataset=train_dataset,

    processing_class=processor.feature_extractor,  # v5: processing_class, not tokenizer

)

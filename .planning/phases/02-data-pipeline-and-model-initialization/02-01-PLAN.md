<objective>
Create `surt/model.py` with model and processor initialization that correctly configures Whisper for Gurmukhi transcription, preventing the catastrophic misconfiguration pitfalls identified in research.

Purpose: Model initialization is the highest-risk single point of failure -- setting language/task wrong means the model outputs English instead of Gurmukhi, making all training useless. This plan ensures language, task, forced_decoder_ids, and generation_max_length are all set correctly on both the processor AND model.generation_config.

Output: `surt/model.py` with `load_model_and_processor()` and `get_mool_mantar_prompt_ids()` functions.
</objective>

  <action>
Create `surt/model.py` with two public functions:

**1. `load_model_and_processor()` -> tuple[WhisperForConditionalGeneration, WhisperProcessor]:**

- Import `BASE_MODEL`, `GENERATION_MAX_LENGTH`, `MOOL_MANTAR` from `surt.config`
- Load processor: `WhisperProcessor.from_pretrained(BASE_MODEL, language="punjabi", task="transcribe")`
- Load model: `WhisperForConditionalGeneration.from_pretrained(BASE_MODEL)`
- Set ALL of these on `model.generation_config`:
  - `language = "punjabi"`
  - `task = "transcribe"`
  - `forced_decoder_ids = None`
  - `max_length = GENERATION_MAX_LENGTH` (448)
- Print confirmation: `[model] Loaded {BASE_MODEL} with language=punjabi, task=transcribe, max_length={GENERATION_MAX_LENGTH}`
- Return `(model, processor)`

**CRITICAL anti-patterns to avoid (from research):**
- Do NOT set language/task only on processor -- must ALSO set on model.generation_config or eval will output English
- Do NOT leave forced_decoder_ids at default -- must explicitly set to None
- Do NOT use `model.config.forced_decoder_ids` -- use `model.generation_config.forced_decoder_ids`

**2. `get_mool_mantar_prompt_ids(processor: WhisperProcessor) -> list[int]`:**

- Tokenize MOOL_MANTAR using `processor.tokenizer(MOOL_MANTAR, add_special_tokens=False).input_ids`
- Return the list of token IDs
- These are passed as `prompt_ids` to `model.generate()` during eval for Gurmukhi vocabulary anchoring

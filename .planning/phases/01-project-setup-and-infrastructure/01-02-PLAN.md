**Steps to verify:**

Step 1: Provision RunPod pod
1. Go to https://www.runpod.io/console/pods
2. Create a spot GPU pod (RTX 3090 or RTX A5000 recommended, both less than $0.30/hr)
3. Select template: "PyTorch 2.7.1 + CUDA 12.6" (or latest PyTorch template)
4. Set environment variable: HF_TOKEN=hf_your_token_here in the pod config
5. Start the pod and SSH in

Step 2: Clone and run setup
```bash
cd /workspace
git clone your-repo-url surt
cd surt
bash setup_pod.sh
```

Step 3: Verify success criteria (all 4 must pass)

Criterion 1 -- Python imports:
```bash
python -c "import transformers, datasets, audiomentations, jiwer, huggingface_hub; print('ok')"
```
Expected: prints "ok"

Criterion 2 -- HuggingFace auth:
```bash
python -c "from huggingface_hub import HfApi; user = HfApi().whoami(); print(f'Logged in as: {user[\"name\"]}')"
```
Expected: prints your HuggingFace username

Criterion 3 -- tmux persistence:
```bash
tmux ls
```
Expected: shows "surt" session. Disconnect SSH, reconnect, run `tmux ls` again -- session should still exist.

Criterion 4 -- GPU config detection:
```bash
python -c "from surt.config import GPU_NAME, BATCH_SIZE, GRAD_ACCUM, EFFECTIVE_BATCH; print(f'GPU: {GPU_NAME}, Batch: {BATCH_SIZE}, Accum: {GRAD_ACCUM}, Effective: {EFFECTIVE_BATCH}')"
```
Expected: prints detected GPU name with correct batch/accum (e.g., RTX 3090 -> batch=8, accum=4, effective=32)
